## Vision / 目指していること

人口減少・財政縮小という現実を隠さず、住民と共に選び取る自治体の姿を作りたい。  
透明性とテクノロジーで、身の丈にあった日本の地域社会をデザインする。

Rather than concealing the realities of population decline and fiscal constraints,  
I believe in designing local governance that is honest, transparent, and technology-enabled —  
so that communities can make informed choices about their own future.
